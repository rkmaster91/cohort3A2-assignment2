# Cohort 3 Assignment 2 (Day 5)

This repository contains my Day 5 assignment from Sheryians Coding School Cohort 3.

In this assignment, I recreated three different landing pages using only HTML and CSS to improve my understanding of layouts, positioning, Flexbox, and styling.

## Projects

### 1. ScaleX

A SaaS/blog landing page with:

* Navigation bar
* Blog cards
* Category buttons
* Hover effects

### 2. GUNAFORCYTER

A restaurant landing page with:

* Product cards
* Chef's special section
* Footer
* Gradient styling

### 3. Reload

A creative agency landing page with:

* Hero section
* Image gallery
* Typography-based design
* Minimal layout

## Tech Used

* HTML5
* CSS3
* Remix Icons

## What I Practiced

* Flexbox
* Positioning
* Gradients
* Hover effects
* Typography
* Building layouts from UI references

## Run Locally

Just open the `index.html` file of any project in your browser.
